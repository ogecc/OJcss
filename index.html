OJcss

GitHub Install Docs

Press start to begin_

OJcss is a minimal pixelated CSS stylesheet — black & green, no JavaScript, no build step. Drop one file into your project and every semantic HTML element gets a retro 8-bit / terminal makeover.

Installation

1. Download

Grab ojcss.css from the repo and link it:

<link rel="stylesheet" href="ojcss.css">

2. CDN (jsDelivr)

<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/gh/ogecc/OJcss@main/ojcss.css">

3. NPM
